In this project, I learned how to build a weather app, implementing React JS and APIs. As basic as it seems, I broadened my knowledge on how APIs work, in which I 
needed to use an API key from OpenWeatherMap and also their API call/request url in order for the website to function. As for React JS, I used their vast library to include
important functions and variables to my project. With that, I also broadened my knowledge on DOM APIs through React JavaScript. Having this as my first official Git project helped
me understand the way git functions and how to use it within VSCode. Also, starting the project with node.js' node packange manager and vite framework gave me understanding on installing JavaScript 
packages, creating the very base of my project which includes the very import package.json.
